# 10. Arithmetic and Comparison

## 10.1 Arithmetic Operations

Arithmetic uses noun compounds or verbs:

### Noun Compound Style

```glossa
α β ἄθροισμα           // "of-a of-b sum" → a + b
α β διαφορά            // "of-a of-b difference" → a - b
α β γινόμενον          // "of-a of-b product" → a * b
α β μερισμός           // "of-a of-b quotient" → a / b
α β ὑπόλοιπον          // "of-a of-b remainder" → a % b
```

### Verb Style (Dual)

```glossa
α βῳ προστίθεται.      // "a to-b adds-itself" → a + b
α ἀπὸ β ἀφαιρεῖται.    // "a from b subtracts" → a - b
α βῳ πολλαπλασιάζεται. // "a by-b multiplies" → a * b
α βῳ διαιρεῖται.       // "a by-b divides" → a / b
```

## 10.2 Comparison

Comparisons use the **genitive of comparison** + comparative adjectives:

| Greek | Meaning | Operator |
|-------|---------|----------|
| μεῖζον + GEN | greater than | `>` |
| ἔλαττον + GEN | less than | `<` |
| ἴσον + DAT | equal to | `==` |
| οὐκ ἴσον + DAT | not equal to | `!=` |
| μεῖζον ἢ ἴσον + GEN | greater or equal | `>=` |
| ἔλαττον ἢ ἴσον + GEN | less or equal | `<=` |

### Examples

```glossa
ξ πέντε μεῖζον
"x than-five greater"
// → x > 5

ξ υῳ ἴσον
"x to-y equal"
// → x == y

ξ μηδενὸς ἔλαττον ἢ ἴσον
"x than-zero less or equal"
// → x <= 0
```

## 10.3 Boolean Operations

```glossa
α καὶ β      // a AND b
α ἢ β        // a OR b
οὐκ α        // NOT a
οὐχ α        // NOT a (before rough breathing)
μή α         // NOT a (in non-indicative moods)
```
