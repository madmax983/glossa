**Applying Facade to `glossa::tools`**
**Tangle:** The `glossa::tools` module exposed its internal structure by making all sub-modules `pub` (e.g., `pub mod runner`, `pub mod cli`). This leaked implementation details and required deep import paths like `glossa::tools::runner::run_file` everywhere.
**Blueprint:** Modified `src/tools/mod.rs` to make all internal sub-modules `pub(crate)` and created a explicit Facade at the bottom using `// --- Facade Re-exports ---`. We explicitly `pub use` only the necessary structures and functions, creating a cleaner, flattened API surface (e.g., `glossa::tools::run_file`).
